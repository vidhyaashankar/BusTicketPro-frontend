# Bus Ticket Booking System

This is a full-stack Bus Ticket Booking System built with React, Express, and SQLite.

## How to Run in VS Code

Follow these simple steps to run the project locally on your machine:

### 1. Prerequisites
Make sure you have **Node.js** (v18 or higher) installed on your system. You can download it from [nodejs.org](https://nodejs.org/).

### 2. Setup
1. Download the ZIP file and extract it.
2. Open the extracted folder in **VS Code**.
3. Open a terminal in VS Code (`Ctrl + ` ` or `Terminal > New Terminal`).

### 3. Installation
Run the following command to install all necessary dependencies:
```bash
npm install
```

### 4. Run the Application
Start the application with a single command:
```bash
npm run dev
```

### 5. Access the Output
Once the server is running, you will see a message like:
`Server running on http://localhost:3000`

Open your browser and navigate to:
**[http://localhost:3000](http://localhost:3000)**

## Features
- **User Authentication**: Register and Login with password validation.
- **Bus Search & Booking**: Search for routes and book seats.
- **Dashboard**: View and manage your bookings.
- **Admin View**: (Optional) Manage all tickets.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Lucide React, Motion.
- **Backend**: Node.js, Express, Better-SQLite3.
- **Database**: SQLite (Local file: `bus_tickets.db`).
